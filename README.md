# Breast Cancer Classification (Top 5 Features)

This project uses Logistic Regression to classify breast tumors as
Benign or Malignant using only the top 5 most important features.

##Datasetlink
link:-https://www.kaggle.com/datasets/neurocipher/breast-cancer-dataset

## Features Used
- concave_points_worst
- radius_worst
- perimeter_worst
- area_worst
- concavity_worst

## Model
- Logistic Regression
- Accuracy: ~96.5%

## Evaluation
- Precision (Malignant): 1.00
- Recall (Malignant): 0.91

## Tools
- Python
- Pandas
- Scikit-learn
- Google Colab
